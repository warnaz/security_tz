    Как работает бот
      №_1
        На фоне каждую минуту проверяются самые популярные rss источники:
          - Кэшированный запрос
          - Фильтрация новостей, чтобы было за последний час
          - Добавление новых данных в Redis

      №_2
        Пользователь нажимает на кнопку "за последний час"
          - Группировка оставшихся источников по чанкам, чтобы не отправлять разом запросы 
          - Отправляются запросы на источники из чанков
          - Фильтрация новостей, чтобы было за последний час
          - Группировка полученных новостей с новостями из Redis
          - Чтобы пользователь не ждал все новости, отправляем ему полученные новости из чанков по мере получения

      №_3
        Пользователь нажимает на кнопку "за сутки"
          - Группировка источников по чанкам, чтобы не отправлять разом запросы 
          - Отправляются запросы на источники из чанков
          - Чтобы пользователь не ждал все новости, отправляем ему полученные новости из чанков по мере получения
